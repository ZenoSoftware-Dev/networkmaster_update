# NetworkMaster (Enterprise Edition)

Profesjonalne, wieloplatformowe narzędzie do diagnostyki, zarządzania i monitorowania sieci. Zaprojektowane dla administratorów IT, którzy potrzebują szybkiego dostępu do kluczowych funkcji w terenie.

## 🚀 Kluczowe Możliwości

### 🛡️ Diagnostyka i Bezpieczeństwo
* **WiFi Security Auditor:** Odzyskiwanie zapomnianych haseł WiFi (Windows: `netsh`, Linux: `nm-connection-editor/root`).
* **Audyt Certyfikatów SSL/TLS:** Natychmiastowa weryfikacja wystawcy (CA) oraz dokładnej daty wygaśnięcia certyfikatu dowolnej domeny.
* **Port Scanner:** Wielowątkowy skaner otwartych portów z detekcją usług.
* **Network Scanner:** Skanowanie całej podsieci (ARP + Ping Sweep) w celu wykrycia podłączonych urządzeń.
* **MAC Vendor Lookup:** Automatyczna identyfikacja producenta urządzenia po adresie MAC.
* **GeoIP Locator:** Geolokalizacja publicznych adresów IP.

### 🌐 Diagnostyka Internetu i Domen
* **DNS Benchmark:** Testowanie opóźnień najpopularniejszych publicznych serwerów DNS w celu znalezienia najszybszego dostawcy dla Twojego łącza.
* **Globalny Test Propagacji DNS:** Błyskawiczne sprawdzanie rekordów i propagacji domen na globalnych serwerach (Google, Cloudflare, OpenDNS, Quad9).
* **WHOIS:** Odczyt danych rejestracyjnych i dat wygasania domen internetowych.
* **Traceroute & Ping:** Klasyczne narzędzia do testowania opóźnień i śledzenia trasy pakietów.

### 🔧 Zarządzanie i Konfiguracja
* **Menedżer Kart Sieciowych:** Szybkie przełączanie trybu interfejsów (DHCP/Static) i ich błyskawiczny restart z poziomu jednego panelu.
* **IP Profile Manager:** Szybkie przełączanie między konfiguracjami (DHCP / Static IP) – idealne dla serwisantów obsługujących wiele lokalizacji.
* **Wake On LAN (WOL):** Baza komputerów z możliwością zdalnego wybudzania.
* **Kalkulator Podsieci (CIDR):** Wyliczanie adresu sieci, broadcastu oraz liczby dostępnych hostów na podstawie maski.
* **Network Repair:** Jedno kliknięcie do resetowania stosu TCP/IP, czyszczenia cache DNS i restartu kart sieciowych.
* **Hosts Editor:** Wbudowany edytor pliku `/etc/hosts` (Linux) lub `C:\Windows\System32\drivers\etc\hosts`.

### 📊 Monitoring Live
* **Globalny Ekran Dowodzenia (Dashboard):** Zaawansowany panel monitorujący w czasie rzeczywistym ping do internetu oraz sumaryczne obciążenie kart sieciowych.
* **Bandwidth Monitor:** Podgląd użycia łącza (Download/Upload) w czasie rzeczywistym.
* **WiFi Signal Dashboard:** Graficzny monitor siły sygnału, kanału i BSSID (działa również na Linux Kernel Realtime).
* **Szyfrowanie Lokalne: Wrażliwe dane (jak hasła, rejestry i logi) są chronione zaawansowanymi algorytmami kryptograficznymi przed nieautoryzowanym dostępem.

### ☁️ Architektura Cloud (Enterprise)
* **Auto-Update (OTA):** Inteligentny system aktualizacji pobierający binarki bezpośrednio z GitHub Releases.
* **Licencjonowanie:** Centralne zarządzanie licencjami (Lemon Squeezy) z obsługą HWID i szyfrowaniem offline.
* **Cross-Platform:** Natywne wsparcie dla **Windows 10/11** oraz **Linux** (Debian/Ubuntu/Mint/Kali).
* **Szyfrowanie Lokalne:** Wrażliwe dane (jak hasła, rejestry i logi) są chronione zaawansowanymi algorytmami kryptograficznymi przed nieautoryzowanym dostępem.
